SQL Schema Visualizer

Free, fast, in-browser database schema visualizer. Paste PostgreSQL, MySQL, SQLite or ANSI CREATE TABLE DDL and get an interactive ER diagram with foreign keys and comments — entirely client-side, nothing leaves your browser.

Features

  • PostgreSQL / MySQL / SQLite / ANSI DDL parsing
  • Interactive ER diagram with foreign-key edges
  • Click-to-navigate tables and fields, column filter
  • SQL comment preservation (off / inline / hover)
  • Shareable diagrams via compressed URL
  • Light and dark themes, collapsible tables
  • 100% client-side — no backend, no signup
